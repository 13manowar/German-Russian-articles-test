# German Article Trainer (PyQt6 + Excel)

Интерактивное приложение для изучения немецких слов с артиклями (**der / die / das**)  
с системой ошибок, фильтрацией уровней и редактированием базы через Excel.

## Описание

Приложение предназначено для тренировки немецких слов с артиклями и помогает:
- быстрее запоминать слова
- автоматически проверять ответы
- работать с реальной базой Excel
- отслеживать ошибки

---

## Основные возможности

### Обучение
- Проверка артиклей: **der / die / das**
- Случайная выдача слов
- Работа со всеми листами Excel

### Фильтрация
- Уровни:
  - A1
  - A2
  - B1
  - B2
  - C1
- Динамическое обновление выборки

### Управление словами
- Добавление новых слов
- Редактирование текущего слова
- Исключение слов из повторения

### Настройки интерфейса
- показать / скрыть артикль
- убрать цвет (монохромный режим)
- скрыть перевод

### Статистика
- ошибки за текущий сеанс
- сброс ошибок

---

## Структура Excel-файла

**Файл:** `Книга 1.xlsx`

| Колонка | Описание |
|--------|----------|
| A | слово (например: der Tisch) |
| B | перевод |
| C | уровень (A1–C1) |
| D | ошибки |
| E | флаг "не повторять" |

## Интерфейс

Приложение создано с использованием **PyQt6 + Qt Designer**

Основные элементы:
- отображение слова
- перевод
- кнопки "артикли":
  - DER
  - DIE
  - DAS
- управление:
  - добавить слово
  - редактировать слово
  - сброс ошибок
- настройки отображения
